О проекте

Парсер вакансий с HeadHunter

Веб-приложение для поиска и анализа вакансий с сайта hh.ru

Возможности
  • Поиск вакансий по ключевым словам
  • Фильтрация по городам
  • Сохранение результатов в базу данных
  • Экспорт в Excel и CSV
  • История поисковых запросов
  • Статистика по найденным вакансиям
Технологии
  • Backend: Python, Flask, SQLAlchemy
  • Парсинг: Requests, BeautifulSoup4
  • Данные: Pandas, OpenPyXL
  • Frontend: Bootstrap 5, jQuery
  • База данных: SQLite
Как это работает
  1. Вы вводите поисковый запрос и город
  2. Парсер отправляет запросы к hh.ru
  3. Собирает информацию о вакансиях
  4. Сохраняет данные в базу данных
  5. Отображает результаты в удобной таблице
  6. Позволяет скачать данные в Excel/CSV
Важно: Приложение предназначено только для образовательных целей. Уважайте robots.txt сайтов и не создавайте излишнюю нагрузку.
