# job_parser
# Парсер вакансий с HeadHunter

Веб-приложение для поиска и анализа вакансий с сайта hh.ru с возможностью экспорта в Excel/CSV.

##  Возможности

### Основные функции
-  Поиск вакансий по ключевым словам
-  Фильтрация по городам
-  Сохранение результатов в базу данных
-  Экспорт в Excel и CSV
-  История поисковых запросов
-  Статистика по найденным вакансиям

### Технические особенности
-  Асинхронный парсинг
-  Кэширование результатов
-  Обработка ошибок и повторные попытки
-  Адаптивный дизайн

##  Технологии

### Backend
- **Python 3.11+**
- **Flask** - веб-фреймворк
- **SQLAlchemy** - ORM для работы с БД
- **Requests** - HTTP запросы
- **BeautifulSoup4** - парсинг HTML
- **Pandas** - обработка данных
- **OpenPyXL** - работа с Excel

### Frontend
- **Bootstrap 5** - адаптивный интерфейс
- **jQuery** - AJAX запросы
- **Bootstrap Icons** - иконки

##  Установка

### Предварительные требования
- Python 3.11 или выше
- pip

### Пошаговая инструкция

**1. Клонировать репозиторий**
**2. Создать виртуальное окружение**
Windows
python -m venv venv
venv\Scripts\activate

Linux/Mac
python3 -m venv venv
source venv/bin/activate
**3. Установить зависимости**
pip install -r requirements.txt
**4. Запустить приложение**
python app.py
